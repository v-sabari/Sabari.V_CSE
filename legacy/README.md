# Legacy files

These files (`blog.html`, `script.js`, `style.css`, `firebase.js`) are from an
earlier plain HTML/CSS/JS version of this portfolio. They are **not** used by
the current Vite + React app (the app's only entry point is `index.html` →
`/src/main.jsx`).

They're kept here for reference only. If you don't need them, it's safe to
delete this folder. If you want to revive the blog page, consider rebuilding
it as a React route instead of a standalone static page.
